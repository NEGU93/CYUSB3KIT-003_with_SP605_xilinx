===============================
CYUSB3KIT-003 with SP605 xilinx
===============================

:Author:  J. Agustin Barrachina
:Version: 1.0 of 2019/02/27 

.. cpp:class:: FX3USB3Connection

The cpp code is a class that enables easy connect and communication with a Cypress CYUSB3KIT-003 and hopefully to other Cypress devices.
The projec was done in the context of communicating with a SP605 Xilinx board but it should be useful for other applications.
